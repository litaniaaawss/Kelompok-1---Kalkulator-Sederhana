# Aplikasi Kalkulator Sederhana
Aplikasi Kalkulator ini merupakan aplikasi yang dibuat dakam bahasa Python. Kalkulator ini bisa dijalankan di terminal/console. Program kalkulator sederhana ini memiliki perhitungan sederhana seperti penjumlahan, pengurangan, perkalian dan pembagian


## Features Aplikasi Kalkulator Sederhana
- Penjumlahan
- Pengurangan
- Perkalian
- Pembagian
